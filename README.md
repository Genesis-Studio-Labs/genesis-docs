# Genesis Studio Docs

Internal documentation site for Genesis Studio, built with Docusaurus and published to GitHub Pages.

## Local development

```bash
npm install
npm start
```

## Production build

```bash
npm run build
npm run serve
```

## Deployment

Deployments are handled automatically by GitHub Actions on pushes to `main`.

The published site URL is:

```text
https://pr0y.github.io/genesis-docs/
```
